# PDF 水印工具

一个快速给 pdf 文件加水印的工具

## 线上地址

[https://zhengpq.github.io/pdf-watermark/](https://zhengpq.github.io/pdf-watermark/)

## 本地运行

clone 本项目然后运行一下命令即可在本地体验

```
npm install

npm run start
```

## 具备功能

- 添加文字水印
- 添加图片水印
- 选择部分页面不加水印
- 自定义水印大小，颜色，密度，旋转角度
- 效果预览
- 上下键切换页面
